

# نظام إدارة الموارد البشرية (HR System) - خطة متكاملة

## نظرة عامة
نظام HR متكامل ثنائي اللغة (عربي/إنجليزي) لشركة متوسطة الحجم، مبني على Supabase مع مصادقة وصلاحيات كاملة.

---

## المرحلة 1: البنية الأساسية والمصادقة

### 1.1 المصادقة وإدارة المستخدمين
- تسجيل دخول/خروج بالبريد الإلكتروني وكلمة المرور
- إعادة تعيين كلمة المرور
- صفحة البروفايل الشخصي

### 1.2 نظام الصلاحيات (Roles)
- أدوار: مدير عام (Admin)، مدير HR، مدير قسم، موظف
- جدول صلاحيات منفصل مع RLS
- كل دور يرى فقط ما يخصه

### 1.3 دعم اللغة العربية والإنجليزية
- نظام i18n متكامل مع تبديل فوري بين اللغتين
- دعم RTL للعربية و LTR للإنجليزية
- جميع النصوص والتسميات بالللغتين

### 1.4 التصميم العام
- شريط جانبي (Sidebar) بالأقسام الرئيسية
- شريط علوي بالإشعارات وتبديل اللغة والبروفايل
- لوحة تحكم رئيسية (Dashboard) بإحصائيات سريعة
- تصميم متجاوب للشاشات المختلفة

---

## المرحلة 2: إدارة الموظفين

### 2.1 بيانات الموظفين
- سجل كامل لكل موظف: الاسم، رقم الهوية، تاريخ الميلاد، الجنسية، الحالة الاجتماعية
- بيانات التواصل: الهاتف، البريد، العنوان
- بيانات العمل: المسمى الوظيفي، القسم، الفرع، تاريخ التعيين، نوع العقد
- رفع المستندات (صورة الهوية، العقد، الشهادات)
- حالة الموظف: نشط، في إجازة، منتهي الخدمة

### 2.2 الهيكل التنظيمي
- إدارة الأقسام والفروع
- تعيين مدير لكل قسم
- عرض شجري للهيكل التنظيمي

### 2.3 البحث والتصفية
- بحث سريع بالاسم أو الرقم الوظيفي
- تصفية حسب القسم، الفرع، الحالة
- تصدير البيانات

---

## المرحلة 3: الحضور والانصراف والإجازات

### 3.1 نظام الحضور
- تسجيل حضور/انصراف يدوي أو تلقائي
- عرض سجل الحضور اليومي والشهري
- حساب ساعات العمل والعمل الإضافي
- تقارير التأخير والغياب

### 3.2 إدارة الإجازات
- أنواع الإجازات: سنوية، مرضية، طوارئ، بدون راتب، أمومة/أبوة
- رصيد الإجازات لكل موظف
- طلب إجازة → موافقة المدير المباشر → موافقة HR
- تقويم إجازات الفريق
- تنبيهات عند اقتراب نفاد الرصيد

---

## المرحلة 4: الرواتب والمالية

### 4.1 مسيّرات الرواتب
- إعداد هيكل الراتب: أساسي، بدل سكن، بدل نقل، بدلات أخرى
- حساب الراتب الشهري التلقائي
- الخصومات: تأخير، غياب، سلف، تأمينات
- الإضافات: عمل إضافي، مكافآت، بدل انتقال

### 4.2 التأمينات والضرائب
- حساب التأمينات الاجتماعية (حصة الموظف والشركة)
- إعداد نسب الخصم
- تقارير التأمينات الشهرية

### 4.3 السلف والقروض
- طلب سلفة → موافقة → خصم أقساط شهرية
- متابعة رصيد السلف

### 4.4 كشوف الرواتب
- كشف راتب تفصيلي لكل موظف
- تقرير مسيّر الرواتب الشهري الإجمالي
- تصدير كشوف الرواتب PDF

---

## المرحلة 5: التوظيف والتدريب

### 5.1 إدارة التوظيف
- إنشاء وظائف شاغرة مع الوصف والمتطلبات
- استقبال طلبات التقديم
- مراحل التوظيف: تقديم → فرز → مقابلة → عرض → قبول
- تتبع حالة كل متقدم عبر لوحة كانبان

### 5.2 إدارة التدريب
- خطط تدريب سنوية
- جدولة الدورات التدريبية
- تسجيل الموظفين في الدورات
- تقييم فعالية التدريب
- سجل تدريب لكل موظف

---

## المرحلة 6: تقييم الأداء

### 6.1 دورات التقييم
- تقييمات دورية (ربع سنوية / سنوية)
- معايير تقييم قابلة للتخصيص
- تقييم ذاتي + تقييم المدير

### 6.2 الأهداف و KPIs
- تحديد أهداف لكل موظف مع نسب إنجاز
- مؤشرات أداء رئيسية (KPIs) لكل قسم
- متابعة التقدم بيانياً

### 6.3 التقارير
- تقرير أداء فردي
- مقارنة أداء الأقسام
- تحليل اتجاهات الأداء عبر الزمن

---

## المرحلة 7: لوحة التحكم والتقارير

### 7.1 Dashboard رئيسي
- إجمالي الموظفين حسب الحالة
- نسبة الحضور اليومي
- الإجازات المعلقة
- إجمالي الرواتب الشهري
- رسوم بيانية تفاعلية (Recharts)

### 7.2 الإشعارات
- إشعارات داخلية لطلبات الإجازات والموافقات
- تنبيهات انتهاء العقود والإقامات
- تنبيهات تقييمات الأداء

### 7.3 التقارير العامة
- تقارير قابلة للتخصيص والتصدير
- إحصائيات دوران الموظفين
- تقارير مالية ملخصة

---

## قاعدة البيانات (Supabase)
الجداول الرئيسية:
- **profiles** - بيانات المستخدمين
- **user_roles** - الصلاحيات (جدول منفصل)
- **departments** - الأقسام
- **branches** - الفروع
- **employees** - بيانات الموظفين الكاملة
- **documents** - مستندات الموظفين
- **attendance** - سجلات الحضور
- **leave_types** - أنواع الإجازات
- **leave_requests** - طلبات الإجازات
- **leave_balances** - أرصدة الإجازات
- **salary_structures** - هياكل الرواتب
- **payroll** - مسيّرات الرواتب الشهرية
- **deductions/additions** - الخصومات والإضافات
- **loans** - السلف والقروض
- **job_postings** - الوظائف الشاغرة
- **applicants** - المتقدمين
- **training_courses** - الدورات التدريبية
- **training_enrollments** - تسجيلات التدريب
- **evaluation_cycles** - دورات التقييم
- **evaluations** - التقييمات
- **goals** - الأهداف
- **kpis** - مؤشرات الأداء
- **notifications** - الإشعارات

جميع الجداول محمية بـ Row Level Security (RLS) مع صلاحيات حسب الدور.
